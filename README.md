# Website for preparing for the ticket exam

## What is it?
The site is designed to prepare for the ticket exam. In the admin panel, you need to add tickets, specifying the option number and a photo of the ticket, then anyone can go to the ticket page and add their answer to it or see the answers of other people. ðŸ‘Œ

## What used?
* Python 3.7+
* Django 3
* CSS 3
* HTML 5
* Bootstrap 4

## How to run?
In order to run this project you need:
1) Installation `requarements.txt` file with the `pip install -r requarements.txt` command
2) Run `Tickets` with the `python manage.py runserver` command
